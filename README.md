# Interactive_Sales_Dashboard
Power BI Project: Sales Performance Dashboard

I recently built an interactive Sales Performance Dashboard using Power BI to analyze sales trends, regional performance, and key business metrics.

The goal of this project was to transform raw sales data into actionable insights through data cleaning, modeling, and visualization.

Key Features

Data Cleaning & Transformation: Processed raw data using Power Query to handle duplicates, correct data types, and structure the dataset for analysis.

Data Modeling & Calculations: Created DAX measures to calculate KPIs such as Total Sales, Total Orders, Total Quantity, and Average Order Value.

Interactive Visualizations: Designed charts including trend analysis, regional performance comparisons, and category distribution.

Dynamic Filtering: Implemented slicers to allow users to filter the dashboard by region, product category, salesperson, and time period.

Insights the Dashboard Can Provide

Monthly sales trends over time

Regional sales performance comparison

Category contribution to overall sales

Key sales KPIs for quick business monitoring
